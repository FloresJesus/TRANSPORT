# 02 - Diseño del Sistema

## Diseño General

Se desarrolló una interfaz moderna tipo fintech, enfocada en simplicidad y rapidez.

## Pantallas Principales

### Login

- Correo electrónico
- Contraseña
- Botón ingresar
- Modo demo

### Dashboard

- Total gastado
- Gráfico de gastos
- Registro de movimientos
- Historial reciente
- Ahorro estimado

## Flujo de Usuario

Ingreso → Registro gasto → Análisis → Recomendación

## Arquitectura Técnica

Frontend:
- Next.js
- React

Visualización:
- Chart.js

Hosting:
- Vercel

Versionamiento:
- GitHub

Base de Datos (opcional):
